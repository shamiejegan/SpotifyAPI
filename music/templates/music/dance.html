Get the most danceable tracks

API Structure

GET api/tracks/dance/[n]

About the API

Danceability is a measure quantifying how danceable a track is based on a combination of musical elements including tempo, rhythm stability, beat strength, and overall regularity. It ranges from a value of 0.0 (least danceable) to 1.0 (most danceable). This API ranks all tracks based on its danceability metric, and returns the n most danceable tracks. The track's name, artist, release date, and duration in milliseconds are returned.
