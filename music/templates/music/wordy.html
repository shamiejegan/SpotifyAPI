Get tracks with the most speech in it

API Structure

GET api/tracks/wordy/[n]

About the API

This API ranks all tracks based on its wordiness metric, and returns the n most wordy tracks. Wordiness is a measure quantifying the presence of spoken words in the track. Typically, values above 0.33 indicate tracks with more speech in it. This includes both music and speech (including rap music). The track's name, artist, release date, and duration in milliseconds are returned.
